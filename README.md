# Insurance Client Manager

This is a simple Python program for managing insurance clients right from your terminal.

You can add new clients, view a list of all clients, search for them, and see a basic report.

## How to Run

1.  Make sure you have Python 3 installed.
2.  Save the code as `insurance.py`.
3.  Open your terminal and run
4.  When asked, enter the password: `admin#22`
5.  Follow the menu options (1-7).

**Note:** This program does not save your data. All added clients will be gone when you close the program.
